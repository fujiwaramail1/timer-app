# Timer App ⏱

シンプルでスタイリッシュなタイマーアプリです。

## 機能

- ⏱ 時間・分・秒を自由に設定
- ⏸ スタート / 一時停止 / リセット
- ➕ +1分ボタンで時間追加
- 🎯 プリセットボタン（30秒〜1時間）
- ⚠️ 残り10秒で警告表示（赤色に変化）
- 🔔 タイマー終了時に音と画面フラッシュ
- ⌨️ スペースキーでスタート/一時停止

## 使い方

`index.html` をブラウザで開くだけで動作します。インストール不要。

## 技術スタック

- HTML / CSS / JavaScript（バニラ）
- Google Fonts (Syne, DM Mono)

## スクリーンショット

ダークテーマをベースにしたミニマルなデザインで、進捗バーとステータスインジケーターを備えています。
